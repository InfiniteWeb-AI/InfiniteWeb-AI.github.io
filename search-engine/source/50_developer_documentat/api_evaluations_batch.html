Create Batch Evaluation

Create a batch evaluation run against a dataset in a given project.

/v1/projects/{project_id}/evaluations:batch

Request body

This endpoint expects a JSON request body that defines which dataset, models, and metrics to use for the batch evaluation. See the examples below for a concrete payload.
