About pet profiles

Create a profile for each cat once, then reuse it whenever you book boarding or grooming. Profiles keep key details together so check-in is faster and we can keep your cats comfortable.

  • Save names, ages, lifestyles, and diet notes.
  • Select multiple cats to start a new boarding booking.
  • Book grooming for a single cat directly from their card.

You can update or deactivate a profile at any time if details change.
