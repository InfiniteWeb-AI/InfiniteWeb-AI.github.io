Ask a security question

Describe your problem clearly and include enough technical detail for others to help.

Summarize the specific security problem or question in one sentence.

Choose the area of security that best fits your question.

Use tags to describe technologies, standards, or attack types involved.

Use Markdown for formatting; omit secrets and private keys.

Question guidelines

  • Explain the context: threat model, environment, and constraints.
  • Include relevant logs or config snippets (sanitized of secrets).
  • Describe what you have already tried and why it did not work.
  • Prefer reproducible examples over screenshots when possible.
  • Avoid asking for help executing attacks; focus on defense and analysis.

Example topics

  • Hardening TLS configurations
  • Threat modeling for web applications
  • Secure authentication and SSO (OAuth2, SAML)
  • Analyzing suspicious network traffic
  • Secure storage of credentials and keys
