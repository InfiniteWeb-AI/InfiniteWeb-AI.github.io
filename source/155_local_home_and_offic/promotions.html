How to apply a promotion code on your quote

Follow these steps on the quote and results pages to make sure your discount is applied correctly.

  1. Start a home move quote from this page or from the Home Removals page.
  2. On the quote form or on the results page (home_results.html), look for the field labelled Promotion code or Discount code.
  3. Enter your chosen code exactly as shown here (for example, MOVE10).
  4. Click the Apply or Update price button beside the promotion field to recalculate your prices with the discount included.
  5. If you used a "Start quote with this promotion" link from this page, your code may already be filled in. Simply confirm the code and apply it on the results page.

You can keep this Offers page open in a separate tab while you complete your quote, so you can double-check the code and conditions at any time.
