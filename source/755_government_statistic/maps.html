Maps

Configure an interactive map by choosing a dataset, geography, and value filters, then save or download your view.

Loading map toolsa0a0a0

1. Choose a dataset

Only active datasets that support map views are shown.

2. Set geography

Use custom selection to choose specific areas.
Hold Ctrl (Cmd on Mac) to select multiple geographies. Leave empty to include all.

3. Choose value and filters

Measures are based on the selected dataset.

4. Map preview

Select a dataset, geography level, and value to see the map preview.

Adjust the map center and zoom to refine the preview extent.

5. Save this map view

Provide a name you will recognize later, such as "State unemployment map".

6. Download data for this map

Downloads are based on the currently active saved map view.
